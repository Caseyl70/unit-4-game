Crystal Collector

Instructions:

You will be given a random number at the start of each round; this is your goal number. Each crystal innitally has a value unknown to you, but clicking on each crystal will show you that crystal's specific value. Your job is to match the goal number exactly using a combonation of the four crystals. The values for the crystals change every time you start a new game. Good luck!
